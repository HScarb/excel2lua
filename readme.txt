# 运行方式2: 使用cmd 运行excel2lua.exe ,可以带一个参数, 表示自定义的文件路径配置文件
## 示例: excel2lua.exe file_path.txt
#
# 文件路径配置: 用Notepad++打开 "文件路径.txt" 进行配置
## 注意不要用windows自带记事本打开
## 路径格式示例
```python
# 井号符号为python中的注释
[   # 开头与结尾处方括号不能少
    ['excels\\【关卡】导表_关卡配置', '导表_副本配置','data\\tollgate'],
    # .. 符号为进入上一层文件夹。此处为相对路径，也可以使用全路径
    ['..\\excel2lua\\excels\\【关卡】导表_副本怪物', '导表_副本怪物','data\\monster'],
    ['excels\\导表_技能数据', '导表_技能','data\\skill'],
    #['excels\\导表_状态数据', '导表_副本配置','data\\status'],
]
```
#
# 该导表器直接读取excel文件, 而不是用OLE方式操作excel
# 存在问题: 对于excel中由计算得出的小数 导出值有略微不精确
## 如 1.2 可能导出成 1.200000001
